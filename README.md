# Event Pricing Calculator

A simple tool for estimating event platform costs based on your visitor count and chosen add-ons.

## Top Features

- **Live Calculation**: Price updates instantly as you move the slider or pick add-ons.
- **Plan Comparison**: Quick feature comparison between Premium and Starter plans.
- **Responsive**: Works on mobile and desktop without a hitch.
- **Dark Mode**: Supports both light and dark themes.
- **Zero Build**: Just open `index.html` in your browser. No `npm install` or build steps needed.

## Tech

- **Logic**: Alpine.js
- **Styling**: Tailwind CSS (CDN)
- **Colors**: Color Preset

## Usage

Clone the repo and open `index.html`.

```bash
git clone git@github.com:mystaline/event-pricing.git
```
